# Prompt Log — 用户输入记录开关

**单文件模式**：`.claude/prompts.log.md` 同时作为状态文件和日志文件。YAML frontmatter 控制开关，markdown 主体是日志内容。

## 开启记录

当用户说"开启"、"on"、"start"、"开始记录"时执行：

```!
if [ -f .claude/prompts.log.md ]; then
  perl -i -0777 -pe 's/(^---\n)(.*enabled:) false/$1$2 true/s' .claude/prompts.log.md
else
  mkdir -p .claude && cat > .claude/prompts.log.md <<'EOF'
---
enabled: true
session_id: ${CLAUDE_CODE_SESSION_ID}
started_at: "$(date -u +%Y-%m-%dT%H:%M:%SZ)"
---

# Prompt Log

> Auto-generated by `sl/prompt-log` hook.
> Format: `[TIMESTAMP] [SESSION_ID] PROMPT_CONTENT`

EOF
fi
```

回复："✅ Prompt 记录已开启，后续输入将自动保存到 `.claude/prompts.log.md`。"

## 关闭记录（保留历史）

当用户说"关闭"、"off"、"stop"、"停止记录"时执行：

```!
perl -i -0777 -pe 's/(^---\n)(.*enabled:) true/$1$2 false/s' .claude/prompts.log.md 2>/dev/null || true
```

回复："🛑 Prompt 记录已暂停，历史日志保留在 `.claude/prompts.log.md`。"

## 彻底删除

当用户说"删除"、"清除"、"clean"时执行：

```!
rm -f .claude/prompts.log.md
```

回复："🗑️ 已删除 `.claude/prompts.log.md`，记录彻底清空。"

## 查看状态

当用户问"记录开了吗"、"状态"时执行：

```!
test -f .claude/prompts.log.md && sed -n '/^---$/,/^---$/{ /^---$/d; p; }' .claude/prompts.log.md | grep '^enabled:' | sed 's/enabled: *//' || echo "DISABLED (no file)"
```

## 设计说明

| 场景 | 文件状态 | 行为 |
|------|----------|------|
| 未启用 | 文件不存在 | hook 直接放行，不记录 |
| 开启 | frontmatter `enabled: true` | hook 追加新记录到文件末尾 |
| 关闭 | frontmatter `enabled: false` | hook 放行，不追加 |
| 删除 | 文件被 rm | 回到默认关闭状态 |

**文件格式**：
```markdown
---
enabled: true
session_id: xxx
started_at: 2026-05-29T08:00:00Z
---

# Prompt Log

> Auto-generated by `sl/prompt-log` hook.
> Format: `[TIMESTAMP] [SESSION_ID] PROMPT_CONTENT`

[2026-05-29T08:01:00Z] [xxx] 用户输入内容
```
