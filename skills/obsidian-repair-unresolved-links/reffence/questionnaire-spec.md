# 问卷模式 · 内容规范

> 本规范适用于 obsidian-repair-unresolved-links skill 中问卷模式的响应生成。

---

## 一、核心定位

问卷模式的本质是**信息消费 → 定制化输出**：

```
用户填写问卷 → 提取关键信息 → 分析约束与偏好 → 生成针对性响应
```

与通用模式（解释概念）不同，问卷模式的目的是**解决用户的具体问题**，响应内容必须与用户的实际情况强绑定。

---

## 二、判断标准

### 问卷模式的识别

满足以下条件即为问卷模式：

1. **源文件包含问题列表** — 段落或表格形式的问题
2. **用户有填写内容** — 至少有一个非空回答
3. **末尾有触发双链** — `[[xxx]]` 指向响应锚点或目标笔记

### 问卷 vs 通用模式的区别

| 维度 | 通用模式 | 问卷模式 |
|------|---------|---------|
| 输入 | 用户一句话请求 | 用户的完整问卷回答 |
| 输出 | 解释性内容 | 针对性方案/建议 |
| 风格 | 知识传授 | 问题解决 |
| 深度 | 概念深潜 | 数据分析 + 定制化 |

---

## 三、内容生成流程

### 步骤 1：解析问卷回答

提取并结构化用户回答：

```
基本信息：年龄、性别、体重、身高...
安全筛查：伤病/限制...
活动水平：运动经验、频率...
目标约束：时间、场地、偏好...
```

### 步骤 2：识别关键数据

**必须提取的数据类型：**

| 类型 | 作用 | 示例 |
|------|------|------|
| 人口学数据 | 建立基线 | 21岁/男/72kg |
| 约束条件 | 方案边界 | 每周3次/每次20分钟/晚上练 |
| 偏好/厌恶 | 方案适配 | 讨厌跑步/喜欢哑铃 |
| 限制因素 | 方案禁区 | 膝关节脆弱/腰痛 |
| 基线能力 | 强度起点 | 俯卧撑15个/深蹲20个 |

### 步骤 3：分析优先级

用户可能提出多个需求，按以下顺序决策：

1. **安全优先** — 伤病/限制是硬约束，不能绕过
2. **习惯先行** — 零基础用户先建立节奏感，不是出成绩
3. **可行性优先** — 方案必须符合用户的时间/场地/自驱力约束

### 步骤 4：生成响应

## 四、响应结构规范

问卷模式的响应**不是固定模板**，但必须包含以下要素：

### 必须要素

- [ ] **基线摘要** — 用户的核心数据一览（3-5行）
- [ ] **关键发现** — 从数据中提炼的洞察（优势/短板/约束）
- [ ] **定制化方案** — 与用户情况匹配的解决方案
- [ ] **追踪目标** — 具体可测量的下一阶段目标

### 可选要素（根据场景）

- 备选方案（如用户条件变化）
- 风险提示（如特定动作的注意事项）
- 下一步行动建议

### 禁止要素

- [ ] **不写通用性内容** — "多运动、少吃油炸食品"这种放之四海皆准的话
- [ ] **不写与约束矛盾的方案** — 用户说晚上没空，就不要写早起计划
- [ ] **不写用户未要求的额外内容** — 用户只问了A，不要扯B

---

## 五、写作技法

### 5.1 数据锚点法

用用户的具体数据作为锚点，不要写"一般来说"：

```
❌ 一般来说，新手应该每周运动3-5次。
✅ 根据你的情况（每周3次、每次20分钟），这个频率是可行的。
```

### 5.2 约束满足法

先承认约束，再在约束内找最优解：

```
你讨厌跑步，所以我没有安排跑步。
你只有20分钟，所以我们不堆动作数量。
你的膝盖容易痛，所以所有腿部动作都以降阶版本为默认。
```

### 5.3 优先级可视化

用表格或列表让用户一眼看到重点：

```
## 你的情况摘要

| 维度 | 数据 | 影响 |
|------|------|------|
| 时间 | 每周3次×20分钟 | 只能做精华动作 |
| 场地 | 有哑铃和弹力带 | 可做抗阻训练 |
| 限制 | 膝关节脆弱 | 腿部动作为降阶版 |
| 目标 | 建立习惯 | 前4周不以成绩为目标 |
```

### 5.4 安全感建立

零基础用户最担心"受伤"和"坚持不下来"，在响应中明确：

```
## 防伤原则（必须记住）

1. 膝盖痛 = 立刻停
2. 热身是必须的，不是可选的
3. 某天做不动不是退步，是正常波动
```

## 六、示例

### 问卷场景：健身规划

**源文件：** 用户填写了健身基线问卷

**用户回答关键数据：**
- 21岁/男/72kg/零运动经验
- 膝关节脆弱易痛、腰痛
- 每周3次、每次20分钟、晚上练
- 有哑铃和弹力带，讨厌跑步

**❌ 禁止的响应（通用性内容）：**
```
# 健身建议

## 一、运动的重要性
运动对健康有很多好处...

## 二、如何开始
新手应该从低强度开始...

## 三、饮食建议
少吃油炸食品，多吃蔬菜...
```

**✅ 正确的响应（针对性内容）：**
```
# 健身规划

## 你的情况

- 力量基础不差（俯卧撑15个），但心肺和习惯是短板
- 最大限制是膝关节——所有腿部动作以降阶版为默认
- 时间很紧（20分钟），所以我们不堆动作数量

## 4周计划

### 第1-2周（适应期）
每次20分钟：
- 热身 3分钟
- 深蹲 3×10 → 膝盖痛就改成靠墙静蹲
- 平板支撑 3×20秒
- 快走 3分钟（替代你讨厌的跑步）

### 第3-4周（进阶试探）
加入哑铃和弹力带，试探膝盖耐受度

## 追踪目标

- 完成率 90%（3/3天完成）
- 膝盖不适次数 0次
```

## 七、质量检查清单

- [ ] 响应中引用的每个数据都来自用户问卷（没有捏造）
- [ ] 方案完全满足用户的约束条件（时间/场地/偏好）
- [ ] 方案考虑了用户的限制因素（伤病/痛点）
- [ ] 没有写任何通用性废话
- [ ] 追踪目标是具体可测量的
- [ ] 响应末尾有下一步行动建议
